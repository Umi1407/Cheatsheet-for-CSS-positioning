Cheatsheet for CSS layout position

Tag Description
<Static> The default that every element gets. Normal position in the document.
<Relative> Similar to static position, except that once the element has taken its place in the normal flow, you can be able to modify its final position.
<Absolute> Takes document out of the flow. It no longer takes space like static and Relative. It is positioned absolutely with reference to the closest parent that has a position relative value.
<Fixed> Places into position and stays on that position even when scrolling.
<Sticky> Causes a navigation bar to scroll with the page until a certain point and then stick to the top of the page.
